# HR Manager - Sistema de Gestión de Empleados

## Descripción

Proyecto de base de datos desarrollado para gestionar información de empleados, departamentos y asistencias de una empresa.

El sistema permite almacenar información, realizar consultas, generar reportes y ejecutar procedimientos almacenados utilizando Oracle SQL y PL/SQL.

## Tecnologías utilizadas

- Oracle Database
- SQL
- PL/SQL

## Funcionalidades

- Creación de tablas y relaciones entre datos.
- Gestión de empleados.
- Gestión de departamentos.
- Registro de asistencias.
- Consultas utilizando JOIN.
- Reportes utilizando COUNT y GROUP BY.
- Procedimientos almacenados con PL/SQL.

## Estructura del proyecto

HR_Manager_Portafolio

SQL
- 01_tablas.sql
- 02_datos_prueba.sql
- 03_consultas_reportes.sql
- 04_procedimientos.sql

## Ejemplos realizados

- Consulta de empleados junto con su departamento.
- Reporte de cantidad de empleados por departamento.
- Procedimiento para aumentar salario de empleados.
